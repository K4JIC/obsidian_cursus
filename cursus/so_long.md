必須パート (Mandatory Part)

以下の項目は、プログラムが機能するために必須であり、評価の基本となります。

- [ ] **リポジトリとMakefile**
    - [x] 課題は、個人のGitリポジトリの`master`ブランチにある最後のコミットで提出されている。
    - [x] `Makefile` が存在し、コンパイルが可能である。
- [ ] **エラーハンドリング**
    - [x] 不正なマップ入力（存在しないファイル、不適切な拡張子、無効な文字など）に対して、`Error\n` に続いて適切なメッセージを表示し、クリーンに終了する。
    - [x] メモリリークがない。
- [ ] **マップの検証**
    - [x] マップファイルが `.ber` 拡張子である。
    - [x] マップが長方形である。
    - [x] マップの周囲がすべて壁（`1`）で囲まれている。
    - [x] `P`（プレイヤー初期位置）が1つだけ存在する。
    - [x] `E`（出口）が1つだけ存在する。
    - [x] `C`（収集物）が1つ以上存在する。
    - [x] `0`, `1`, `C`, `E`, `P` 以外の文字が含まれていない。
    - [x] **（重要）** プレイヤーの初期位置から、すべての収集物を集めて出口まで到達可能な有効な経路が存在する。
- [ ] **ゲームプレイと機能**
    - [x] MiniLibX (mlx) を使用してグラフィックが表示される。
    - [x] `mlx_put_image_to_window` 関数を使用して画像が描画されている（ピクセル単位の描画は不可）。
    - [x] `W`, `A`, `S`, `D` キーでプレイヤーが移動できる。
    - [x] プレイヤーが壁を通り抜けられない。
    - [x] すべての収集物を集めた後のみ、出口に到達してゲームが終了する。
    - [x] `ESC` キーを押す、またはウィンドウの赤い十字ボタンをクリックすると、プログラムがクリーンに終了する。
    - [x] プレイヤーの移動回数がシェル（ターミナル）に表示される。

ボーナスパート (Bonus Part)

必須パートがすべて完了し、完璧に動作している場合にのみ評価されます。

- [ ] **追加機能**
    - [ ] 必須要件以外に、敵キャラクター、アニメーション、ゲームオーバー画面などの追加機能が実装されている。
    - [ ] シェルではなく、ゲームウィンドウ内に移動回数が表示されている。
    - [ ] アニメーションまたは複数の画像が表示される。
