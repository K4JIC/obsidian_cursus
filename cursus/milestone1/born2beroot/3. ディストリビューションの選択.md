#### MAC
AppArmor : debian系統に搭載されているセキュリティソフト。名前ベースのアクセス制御で、LSM（Linux Security Module）を用いて実装されている仕組み。
LSMとはLinuxカーネルのフレームワークで、シェルより低レイヤで動いてアクセス制御するシステム。
（/etc/apparmor.d/アプリ名）AppArmorはアプリごとにユーザーのアクセス権限を設定する。これはファイル所有者や管理者が付与するパーミッションよりも優先され、管理者すら読み込み不可なファイルを生成できる。これを、パーミッション設定などの「任意アクセス制御（DAC : discretionary access control）」と比較して「強制アクセス制御（MAC : Mandatory Access Control）」という。
同様のMAC機能を提供するセキュリティソフトとしてSELinuxがある。こちらはrhel, fedra, rockyディストリなどに標準搭載されており、ファイル名だけを参照するAAと異なり、ファイル形式やメタデータも参照してアクセスを制御する。

#### パッケージ管理ツール
dpkg ->apt ->apatitude
- **dpkg (debian package)**
  基本的なパッケージ管理ツール。パッケージのインストール、展開、設定、削除、リスト表示などができる。
- **apt (advanced package tool)**
  進化したパッケージ管理ツール。dpkgに加えてapt-get機能（パッケージの更新、パッケージの依存すべてインストール）、apt-cache機能（パッケージ検索、詳細情報の表示、等）が追加されている。
- **aptitude**
  GUI, CUIから操作できるaptの進化形態。aptに加えて各バージョンへのかんたんなアクセス、sudoの省略（代わりに管理者パスワードを訊ねる）、などの機能がある。
https://www.debian.org/doc/manuals/debian-faq/pkgtools.ja.html
