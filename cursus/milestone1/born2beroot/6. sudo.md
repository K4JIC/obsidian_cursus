## 設定
### ログファイル
```sudoers
Default iolog_dir 
```
設定ディレクティブ,記録される情報,データの形式,主な用途,関連ツール
iolog_dir,セッションの入力/出力の完全な記録 (I/Oログ、アーカイブ)。どのコマンドが実行され、その結果どんな出力があったか。,専用のバイナリ形式,sudoreplay によるセッションの再生。,sudoreplay
logfile,実行されたコマンドのメタデータ（監査ログ）。誰が、いつ、どのホストで、どのコマンドを実行したか（成功/失敗）。,テキスト形式,システムの監査、監視、概要の把握。,"cat, grep などの標準ツール"
### TTYとは
テレタイプライター
```
Default !requiretty
```
これが有効だとコマンドライン以外からのsudo入力（cron, ssh）を受け付けないので無効にしておく。
